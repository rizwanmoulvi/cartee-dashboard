Accept MNEE stablecoin payments in WooCommerce.

== Description ==

A payment gateway for WooCommerce that enables MNEE stablecoin payments with wallet redirect functionality.

== Installation ==

1. Upload plugin files to `/wp-content/plugins/woocommerce-MNEE-gateway/` 
2. Activate the plugin
3. Go to WooCommerce -> Settings -> Payments
4. Enable "MNEE Stablecoin"
5. Add API Key from WooCommerce merchant dashboard on Cartee.xyz/dashboard
5. Click "Save Changes"
6. Click the 'Connect' button beside API key input box
